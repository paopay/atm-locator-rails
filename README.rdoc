# Chase ATM Locator

## https://polar-cove-7405.herokuapp.com/

## How to run from terminal:
```
git clone https://github.com/paopay/atm-locator-rails.git
cd atm-locator-rails
bundle install
rails s
```
Then go to browser and go to localhost:3000
